# Employee API Project

This project fetches employee data from a public REST API and stores it in a CSV file.

## 🔧 Technologies Used
- Python
- Requests Library
- CSV Handling
- Git & GitHub

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the script:
python employee_api.py

## 📂 Output
The script creates a CSV file containing employee data.

## 🚀 Features
- Fetch data from API
- Store in CSV
- Beginner-friendly project
